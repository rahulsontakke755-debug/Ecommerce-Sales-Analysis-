Data Cleaning

  • ShippingCost has a small number of missing values (~1%) - we fill these with 0, consistent with free-shipping orders above the threshold.
  • Rating is naturally missing for Returned/Cancelled orders (no rating given), so we leave those as NaN.
  • We create a Month column for time-series aggregation.
python 
df['ShippingCost'] = df['ShippingCost'].fillna(0)
df['Month'] = df['OrderDate'].dt.to_period('M').astype(str)
df['Weekday'] = df['OrderDate'].dt.day_name()

# For revenue analysis, focus on non-cancelled orders (Cancelled orders generate no revenue)
df_valid = df[df['OrderStatus'] != 'Cancelled'].copy()
print("Rows after excluding cancelled orders:", df_valid.shape[0], "of", df.shape[0])

RFM Customer Segmentation

RFM = Recency (days since last order), Frequency (number of orders), Monetary (total spend). We score each customer 1-4 on each dimension and combine into simple segments to identify our most valuable customers.

python 
snapshot_date = df_valid['OrderDate'].max() + pd.Timedelta(days=1)

rfm = df_valid.groupby('CustomerID').agg(
    Recency=('OrderDate', lambda x: (snapshot_date - x.max()).days),
    Frequency=('OrderID', 'count'),
    Monetary=('NetRevenue', 'sum')
).reset_index()

rfm['R_Score'] = pd.qcut(rfm['Recency'], 4, labels=[4,3,2,1]).astype(int)
rfm['F_Score'] = pd.qcut(rfm['Frequency'].rank(method='first'), 4, labels=[1,2,3,4]).astype(int)
rfm['M_Score'] = pd.qcut(rfm['Monetary'], 4, labels=[1,2,3,4]).astype(int)
rfm['RFM_Total'] = rfm['R_Score'] + rfm['F_Score'] + rfm['M_Score']

def segment(row):
    if row['RFM_Total'] >= 10:
        return 'Champions'
    elif row['RFM_Total'] >= 8:
        return 'Loyal Customers'
    elif row['RFM_Total'] >= 6:
        return 'Potential Loyalists'
    elif row['RFM_Total'] >= 4:
        return 'At Risk'
    else:
        return 'Lost'

rfm['Segment'] = rfm.apply(segment, axis=1)
print(rfm['Segment'].value_counts())

Key Insights & Recommendations

Insights: - Sales peak sharply during the Oct-Nov festive season, roughly 2-2.5x an average month, confirming the importance of festive-period inventory and marketing planning. - Electronics and Fashion are the leading revenue categories, while Groceries drives high order frequency at lower average order value. - UPI and Cash on Delivery remain the dominant payment methods, reflecting typical Indian e-commerce buyer behavior. - The return rate (~9%) is a meaningful cost center, concentrated in certain categories - worth deeper investigation (e.g., size/fit issues in Fashion). - RFM segmentation shows a small "Champions" segment contributing a disproportionately large share of revenue — a strong candidate for loyalty/retention campaigns. - Higher discount tiers increase order volume but reduce average order revenue — useful for evaluating promotional ROI.

Recommendations: 1. Increase inventory and ad spend ahead of Oct-Nov; consider early-bird festive campaigns. 2. Launch a loyalty program targeted at the "Champions" and "Loyal Customers" RFM segments. 3. Investigate root causes of returns in the highest-return categories to reduce reverse-logistics costs. 4. Test moderate discount tiers (10-15%) as they appear to balance volume and revenue better than steep discounts. 5. Expand regional marketing in states outside the current top performers to diversify revenue.
